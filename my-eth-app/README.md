## 环境变量设置

项目需要以下环境变量才能正常运行：

1. 复制 `.env.example` 文件并重命名为 `.env`
2. 填入您的私钥和 API 密钥：
   - `PRIVATE_KEY`: 您的以太坊钱包私钥
   - `INFURA_PROJECT_ID`: Infura 项目 ID
   - `ETHERSCAN_API_KEY`: Etherscan API 密钥

**注意：** 请勿将包含真实私钥和 API 密钥的 `.env` 文件提交到版本控制系统中！
